# FloatPlayer

macOS用の、常に最前面に浮くフローティングプレイヤーです。YouTube・写真(スクリーンショット等)・保存済み動画をウィンドウの透明度を調整しながら表示できます。

## できること

- YouTubeを常時最前面のウィンドウで再生(公式のIFrame Player APIを使用。ダウンロードは行いません)
- 写真・スクリーンショットの表示(ドラッグ&ドロップ対応)
- 保存済み動画ファイルのループ再生(ドラッグ&ドロップ対応)
- ウィンドウ全体の透明度調整・クリックスルー
- 再生開始で自動的にUIを隠し、メニューバーのアイコンから操作を復帰
- YouTube Data API v3を使ったチャプター抽出とジャンプ(概要欄のタイムスタンプから自動抽出)

## ビルド・起動

Xcodeは不要です。Swiftのコマンドラインツールのみで動きます。

```sh
./build_app.sh
open FloatPlayer.app
```

## チャプター機能を使う場合

1. [Google Cloud Console](https://console.cloud.google.com/)でプロジェクトを作成
2. 「YouTube Data API v3」を有効化
3. APIキーを作成(アクセス範囲は「一般公開データ」)
4. アプリのYouTube入力欄の下にあるAPIキー欄に貼り付け(ローカルに保存されます)

## 操作

- 上部のセグメントでYouTube/写真/動画を切り替え
- 下部のスライダーでウィンドウ全体の透明度を調整
- メニューバーのアイコンから「パネルを表示」「クリックスルー」「UIを表示/隠す」「チャプター」「終了」を操作
